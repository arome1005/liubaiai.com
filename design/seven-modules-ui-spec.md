## 5. 落笔（提示词库 / 指令设定 / 模板 / 笔感 / 词典）

**产品语义（权威）**：

- **「落笔」**：顶栏七模块之一，**仍是「落笔」**——指 **本节下列全部能力域**（决定 AI 产出是否贴合作品气质等）。**不是**全屏正文编辑器；**也不是**「锦囊」的同义词。
- **「锦囊」**：**仅**为原「**圣经**」（*story bible* 那类结构化设定）在 **界面上的改名**；与路由 `/bible`、技术类型名 `Bible*` 对应。**落笔 ⊄ 锦囊**、**锦囊 ⊄ 落笔**——前者是模块与 §5 总域，后者是 **其中一脉**（旧圣经子域）的展示名。
- **「写作」**：全屏 / 沉浸正文编辑，**独立页面**（`/work/:workId`，`EditorShell`），**默认主路径**：留白 **新建作品成功 → 直接进入** 该页；亦可被推演 / 流光 / 生辉等 **关联跳入**。语义见 **§9**，勿与「落笔」混用。

**命名补充**：除「锦囊」替代旧「圣经」外，其余 §5 能力在文案中仍归在 **「落笔」** 模块下表述；技术侧路由可继续用 `/bible` 等，无需把「落笔」整体改名为「锦囊」。


| 能力块                  | 规格                                                                           |
| -------------------- | ---------------------------------------------------------------------------- |
| **角色 / 世界观设定（原「圣经」→ UI「锦囊」）** | 结构化存储（字段级，而非纯长文）。例如实体「主角」可挂：性格、技能、过往经历等；检索或输入「主角」时能 **自动关联** 相关条目，供装配器注入上下文。 |
| **指令模板**             | 预设高质量 Prompt 模板，如：打斗描写、环境渲染、情感爆发等；可选按体裁/场景分类，一键插入或作为「生辉」侧栏快捷项。               |
| **私有化调教（笔感）**        | 用户上传自己认可的 **名家段落**（或本书已写片段），作为风格样本；生成侧用于约束语气、节奏、修辞偏好（与纯设定类条目互补）。            |
| **全局词典**             | 本书专有名词表（如：灵石、筑基期、宗门名）；**与「生辉」强关联**，装配或后处理时用于 **约束命名与用语**，避免模型乱造术语。           |



| 维度          | 规格                                                                                                   |
| ----------- | ---------------------------------------------------------------------------------------------------- |
| **路由**      | `/work/:workId/bible`（及未来拆分的同作品子路由，如模板库、笔感样本、词典页）                                                    |
| **Path 参数** | `**workId`**（必填，UUID 风格字符串）                                                                          |
| **数据依赖**    | `getWork(workId)`、锦囊条目（结构化 Markdown/JSON，见 `BiblePage`）；扩展：模板表、风格样本、术语表                              |
| **顶栏主导航**   | 「落笔」：进入 **落笔模块**（当前实现常落在 `/work/{id}/bible` 等子路由；与「锦囊」页重叠时见上：锦囊仅为设定子域展示名）。无 `workId` 时链 `/library` 并提示 |
| **作品子栏**    | **写作** `/work/:workId`（**默认主路径**：留白新建作品成功后进入）· **概要** · **锦囊**（原圣经）· **作品库**                        |
| **子栏展示**    | 书名、`wordCount` 合计、进度游标章节标题                                                                           |
| **页面顶栏**    | 有（非 hub）：落笔相关子页可显示「落笔」+ `workId` 截断；**锦囊**子页亦属落笔模块下的分区                                                                       |


**样式建议**


| 区域     | 建议                                                |
| ------ | ------------------------------------------------- |
| 锦囊侧栏列表 | 与编辑器侧栏统一 `--shell-right-w` 或独立 `min(280px, 32vw)` |
| 表单字段   | `label` + `input/textarea`，间距垂直 `0.75rem`         |


**可实现性**：✅ 落笔域内「锦囊/原圣经」结构化设定骨架已有；模板 / 笔感样本 / 词典及与生辉装配链路的 **完整闭环** 为增量（数据模型 + UI + 装配器读取顺序）。

## 9. 本书正文编辑（「写作」/ 沉浸写作）— 与 §5「落笔」的区分

> **曾误写（已纠）**：旧版把「落笔」与全屏正文页绑死，或把「锦囊」与「落笔」划等号。**纠正**：**「落笔」** = §5 模块名与能力总域；**「锦囊」** = 仅原「圣经」设定类 **UI 改名**；**「写作」** = 独立全屏正文页。**默认主路径**：留白 **新建作品成功 → 直接进入** `/work/:workId`（见 §5 表格「作品子栏 · 写作」）。

| 路由                      | 壳                                  | 说明 |
| ----------------------- | ---------------------------------- | --- |
| `/work/:workId`         | `EditorShell`                      | **写作** / 沉浸写作：章节树 + 正文 + 右侧 AI。**勿把整个页面称为「落笔」**（落笔是七模块之一，≠ 本页）。 |
| `/work/:workId/bible`   | `AppShell` + 作品子栏              | **锦囊**（原圣经）设定编辑；属 **落笔模块** 下的子域/路由，**≠ 落笔二字本身**。 |
| `/work/:workId/summary` | `AppShell` + 作品子栏              | **概要**。 |

生辉 / 问策 / 流光 **草稿合并** → 进入 **写作页** `/work/:id`；与顶栏是否点「落笔」无关。
