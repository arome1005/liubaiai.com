# 留白写作

网文向长篇写作工具：作品 / 章节、自动保存、导出与备份。**当前 Web 端数据在浏览器 IndexedDB**（非本地文件）；存储通过 `WritingStore` 抽象，桌面版可换 SQLite，见 `docs/技术说明.md`。完整路线见 `docs/路线图.md`。

## 开发

```bash
npm install
npm run dev
```

浏览器打开终端提示的本地地址。需 **Node 18+**。

## 本期已实现

- 作品列表、新建 / 删除作品；**从 .txt / .md / .docx 导入为新作品**（见作品库说明）
- **卷 / 篇**分组、章节 CRUD、**拖拽**排序、上移 / 下移
- 按章编辑（Markdown 纯文本）、防抖自动保存；**本地草稿**与关闭页提示；**今日字数**增量
- 本章 / 全书字数（非空白字符）、章 `**wordCountCache`** 加速全书统计
- 写作进度游标（章节行 ◎ 设为进度）
- 章内查找 / **替换**（下一处、全部）；**全书搜索**（全书 / **仅进度游标之前**）；**章节历史**（切换章 / 手动保存 / 导出与整库备份前自动快照，可恢复；备份 zip 含快照）
- 导出本章 / 全书：`**.txt`**（UTF-8 带 BOM）、`**.docx`**（Word）、`**.md**`；**LF / CRLF** 换行（设置）
- 设置：字号、**浅色 / 深色主题**、**存储配额**（用量条与预警）、IndexedDB **占用估算**、诊断模式、**醒目数据说明**；**整库 zip 备份**、**覆盖恢复**与**合并导入**（含参考库数据）
- **稳固性**：章节快照 **每章最多 50 条、超 7 天自动删**；编辑器主栏滚动贴边、正文 **line-height 1.8**；关闭 / 刷新时若仍有未落盘写入会 **beforeunload** 提示
- **参考库（3.1～3.8）**：`/reference` 百万字级 **.txt**、**分块** + **倒排索引**、**全文搜索**、阅读器 **按需加载段落**、**摘录** 与 **全局标签**（多标签、筛选）、摘录可选 **关联创作章节**、与写作 **进度游标** 一致的摘录过滤；编辑器 **灵感便签**；备份/合并含标签与关联；**Schema v8**；`**embeddings` 预留**
- **下一迭代**：**【第 4 组：AI 辅助（圣经护栏、流式输出）】**，见 `docs/路线图.md` 文末「下一步」
