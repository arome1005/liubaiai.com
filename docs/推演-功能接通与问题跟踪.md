## 3.1 缺口任务清单（需开发；不含“已写好只差接线/测试”的项）

> 用户口径：要做到「产品说明书终态」，并且 **云端同步**；语音 **不上传音频**，仅保存（可选：本地转写得到的）**文本**，效果与打字一致。
>
> 说明：
>
> - 本清单只收录 **确实缺代码/缺数据结构** 的项。
> - 像「一致性扫描/规则扫描」这类 **库层已实现** 的，已列在 §3，不在本节重复。

### A. 推演（`/v0/tuiyan`）缺口（需开发）

- **推演数据持久化**：把「对话改纲 / 文策 / 细纲」等从纯内存状态落到数据层（IndexedDB + Supabase Hybrid），与 `workId` 绑定，刷新不丢。
- [~] **大纲编辑写回**：卷/章标题编辑、节点状态、删除/重排等操作接 `repo.update`* 并持久化；替换 `defaultValue` 型“伪编辑”。
  - 卷/章：标题、摘要真实写回（`updateVolume`/`updateChapter`），替换 `defaultValue`。
  - 节点状态：草稿/打磨/锁定等写回与持久化（通过 `TuiyanState.statusByNodeId`，与 `workId` 绑定）。
  - [~] 删除卷/章：接 `repo.delete`* 并刷新树（含级联/保护策略）。
    - 左侧大纲树「删除」已接 `deleteVolume/deleteChapter` + 刷新。
    - 删除确认弹层（避免误删）。
    - 被删节点的选中回退、权限/保护策略补齐。
  - [~] 重排：卷/章排序与 `reorderChapters`（以及卷排序策略）接通。
    - 左侧大纲树「上移/下移」已接通：章用 `reorderChapters`；卷用 `updateVolume(order)` 交换。
    - 拖拽排序（可选）、跨卷移动（可选）、边界禁用与更细 UI 反馈。
- **「标记定稿」真实字段**：从“启发式推断 finalized”升级为“用户显式标记”的字段（并用于进度计算与筛选）。
- **左侧搜索大纲**：实现过滤/跳转定位（至少支持按标题/关键词）。
- **新建卷**：接通 `createVolume` + UI 刷新（当前按钮无逻辑）。
- **导图（可编辑）**：用画布/图流库实现可编辑导图（节点/连线/布局），并提供导出图片（PNG/SVG）与持久化。
- **推演 → 落笔/问策联动闭环（终态）**：
  - 三分支/文策一键写入写作侧栏草稿/或光标插入（按说明书链路）。
  - 一键“去问策跟进”带引用材料/预填新会话（按说明书）。
- **关联藏经（终态）**：替换 `mockLinkedBooks`，实现“关联书籍/浏览藏经/应用到当前大纲”的真实数据与路由联动。
- **场景（scene）实体与节点类型**：支持“场景”作为独立节点（非仅卷/章树），并定义其与章的归属/引用关系。

### B. 流光（`/inspiration`）缺口（需开发）

- **碎片详情/编辑弹层（终态）**：标题、来源、URL、类型、标签、所属作品/集合、收藏/私密/归档开关、完整编辑。
- **收藏/私密/归档落库（云同步）**：
  - 将“收藏”从本地 `favoriteIds` 升级为碎片字段（Supabase 同步、RLS 下可见）。
  - `isPrivate` / `archived` 字段 + 筛选与默认隐藏策略（云同步口径明确：RLS 限制即可；不上传音频）。
- **关联体系（终态）**：碎片关联作品/人物/情节（至少先做关联作品；人物/情节与推演/锦囊联动需 schema）。
- **导入/导出（终态）**：碎片/集合导出（JSON/ZIP）+ 合并导入；与 `exportAllData/importAllDataMerge` 兼容。
- **批量管理（终态）**：多选、批量打标签、批量移动集合、批量删除/归档。
- **多视图补齐（终态）**：瀑布流/密铺（如说明书要求保留）与密度控制。
- **转入写作（三方式）终态闭环**：章末追加/光标插入/侧栏草稿合并三路全量打通（含冲突提示、回跳保持状态）。
- **语音捕捉（文本终态）**：
  - 录音不上传，仅在本机完成转写得到文字（或用户手动补充文字），最终写入碎片 `body`。
  - 备份/合并/云同步仅同步转写文本与元数据，不同步音频文件。
- **书签/引用的 URL 预览（终态）**：保存 URL 并做基础元信息抓取与展示（标题/站点/摘要），纳入备份/合并/云同步。
