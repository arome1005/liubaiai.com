# 留白 — 作品库门户（v0 目标态）

> 路由：`/library`  
> 口径：**最终 v0 对齐 UI + 全功能最终态**（不写“待开发/占位”）。

---

## 1. 页面定位

- **一句话目的**：集中管理所有作品，并作为进入全 App 的起点页面与作品上下文切换中心。
- **主要用户任务（从高频到低频）**：
  - **创建/导入作品**：把一个“写作项目”建立起来，并完成初始元信息（标题、简介、状态、标签、封面）。
  - **快速进入创作**：一键进入 **写作** 页继续写正文；或从作品卡直接进入推演/锦囊/概要。
  - **作品管理**：搜索、筛选、排序、视图切换；编辑元数据；归档/删除；导出与备份。
  - **全局概览**：通过统计区快速判断创作盘子（作品数、章节数、总字数、连载/完结/归档分布）。

---

## 2. 入口与导航

### 2.1 进入本页

- **主入口**：Hub 顶栏“留白”。
- **默认入口**：当用户处于 Hub 体系时，“回首页”默认回到作品库（通常为 `/library`）。
- **从其它页面返回**：浏览器 Back；或顶栏品牌区返回。

### 2.2 从本页离开（去哪里）

- **去写作（继续写作）**：
  - **卡片主体点击**：进入 `/work/:id` 并定位上次写作进度（`progressCursor`）。
  - **卡片悬浮快捷按钮**：“继续写作”同上，但显式表达意图。
- **去推演**：卡片悬浮快捷按钮“推演” → `/logic`（预选当前作品）。
- **去锦囊**：卡片悬浮快捷按钮“锦囊” → `/work/:id/bible`。
- **去概要**：卡片悬浮快捷按钮“概要” → `/work/:id/summary`。
- **去设置**：顶栏设置入口 → `/settings`。

### 2.3 状态保留与深链约定（最终态要求）

- **作品上下文**：
  - 用户在本页进入任意作品后，应用会记住“最近作品”上下文（用于 Hub 顶栏作品上下文区，以及其它模块默认选中）。
  - 从本页跳转到推演/问策/藏经等需要作品上下文的模块时，应携带“优先该作品”的语义（路由参数或 `navigate state`，实现方式不限，但验收效果一致）。
- **本页 UI 状态**：
  - 需要持久化：视图（网格/列表）、排序、筛选条件、搜索词。
  - 返回本页时：恢复上次状态；列表滚动位置与焦点行为按 v0 约定执行。

## 3. 信息架构（v0 目标 UI）

> 下述区块按 v0 的视觉顺序描述；不同屏宽会响应式换行，但区块语义与交互入口不变。

### 3.1 顶部标题区（Header）

- **标题**：页面名称（例如“我的作品”）。
- **说明**：一行副标题（低噪、强调目的）。
- **主操作**：
  - **导入作品**（次要按钮）
  - **新建作品**（主按钮）

交互要求：
- 按钮必须有 loading/disabled 与明确文案（如“导入中…”）。
- 主操作在空态与非空态都需要可见。

### 3.2 统计区（四张统计卡）

统计口径与当前产品说明书一致（最终态仍可在 v0 UI 内实现）：
- **作品总数**
- **有章节作品数**（或“连载中”，最终以 v0 定义为准）
- **总字数（万字）**
- **章节总数**

交互要求：
- 若 v0 定义卡片可点击：点击卡片应直接生成等价筛选（并在工具栏可见）。
- 卡片 tooltip 解释口径（例如：字数统计范围、是否包含草稿/未发布章节等）。

### 3.3 工具栏（Toolbar）

- **搜索框**：作品标题 + 标签（可扩展到简介）。
- **排序**：最近编辑 / 创建时间 / 标题 / 字数 / 章节数（最终以 v0）。
- **视图切换**：网格 / 列表。
- **筛选**：
  - **状态**：连载中 / 已完结 / 归档
  - **内容**：是否有章节
  - **标签**：多选

交互要求：
- 所有筛选组合都必须可视化展示，并可“一键清除全部”。
- 工具栏状态与统计卡点击联动（统计卡触发的筛选要回显在工具栏）。

### 3.4 作品列表区（Grid / List）

#### 3.4.1 网格卡片（Work Card）

卡片包含：
- 封面（可上传）或色块占位
- 标题（截断）
- 标签 chips
- 字数、章节数、更新时间
- 悬浮快捷入口：继续写作 / 推演 / 锦囊 / 概要
- 更多菜单：编辑/封面/导出/归档/删除

#### 3.4.2 列表行（Work Row）

列表视图信息与卡片等价，但布局横向展开并更利于批量管理：
- 左：封面缩略 + 标题 + 标签
- 中：字数/章节/更新时间
- 右：快捷入口 + 更多菜单

### 3.5 空态区（No Works）

当作品数为 0：
- 主文案：鼓励开始创作
- 主按钮：新建作品
- 次按钮：导入作品（并明确支持格式）
- 若演示包存在：提供“一键添加演示包”（是否保留以 v0 为准）

## 4. 功能清单（角落级）

> 按“用户可感知的原子动作”写清：位置、前置条件、执行过程、结果与反馈、失败与提示。

### 4.1 新建作品（WorkFormModal）

- **入口**：Header 右侧“新建作品”按钮。
- **前置条件**：无（在 Hub 内始终可用）。
- **字段（与现有产品说明书一致，最终态也需保留）**：
  - **作品名称**（必填）
  - **作品简介**（可选）
  - **作品状态**：连载中 / 已完结 / 归档（单选 chip）
  - **作品标签**：
    - 三组预设 chip（平台定位/题材类型/创作类型）可多选
    - 支持自定义输入追加标签
  - **同人提示**：当标签包含“同人/同人衍生”等时展示版权提示（含确认动作）
  - **AI 提示**：提示标签会影响 AI 侧栏风格与节奏（与全局 AI 注入口径一致）
- **提交与反馈**：
  - 成功：创建作品 → 作品库出现新卡片 → **自动进入写作页** `/work/:id`（沉浸/zen 模式由 v0 定义；**主路径**）。
  - 失败：表单内字段校验提示；写回失败提示可重试（不产生半成品）。

### 4.2 导入作品（importWorkFromFile）

- **入口**：Header 或工具栏“导入作品”。
- **支持格式**：`.txt` / `.md` / `.markdown` / `.docx`
- **流程（最终态）**：
  - 选择文件 → 解析 → 自动切章（按章回标题）→ 创建作品与章节 → 返回作品库并高亮新作品。
  - 可选：导入后自动进入 **写作** 页并定位第一章（按 v0 交互）。
- **失败策略**：
  - 解析失败：提示原因（格式/损坏/权限）与替代方案（转 txt/md）。
  - 切章失败：回退为单章导入，并提示如何手动分章。
  - 写回失败：不产生半成品；提供重试与取消。

### 4.3 搜索 / 排序 / 筛选 / 视图切换

#### 4.3.1 搜索

- **入口**：工具栏搜索框。
- **范围**：标题 + 标签（可扩展简介）。
- **交互**：输入即时过滤；Esc 清空；无结果展示“无匹配”空态并提示清空筛选/搜索。

#### 4.3.2 排序

- **入口**：工具栏排序下拉。
- **选项**：最近编辑 / 创建时间 / 标题 / 字数 / 章节数（最终以 v0）。
- **交互**：切换排序不改变筛选/搜索；支持升降序（若 v0 有）。

#### 4.3.3 筛选

- **状态**：连载中 / 已完结 / 归档。
- **内容**：是否有章节。
- **标签**：多选标签。
- **回显**：筛选条件以 chip/徽标展示，可逐条清除或一键清除全部。

#### 4.3.4 视图切换（网格/列表）

- **入口**：工具栏视图切换。
- **规则**：切换不改变筛选/排序/搜索；记忆上次选择（本地持久化）。

### 4.4 作品卡片动作

#### 4.4.1 进入写作（继续写作）

- **入口**：卡片主体点击 / 悬浮“继续写作”。
- **结果**：
  - 打开 `/work/:id`，并定位 `progressCursor`（或默认章节）。
  - Hub 顶栏作品上下文更新为该作品。

#### 4.4.2 进入推演

- **入口**：悬浮“推演”。
- **结果**：打开 `/logic` 并预选该作品（作品/章选择器默认选中）。

#### 4.4.3 进入锦囊 / 概要

- **入口**：悬浮“锦囊”“概要”。
- **结果**：打开对应子页，并保持作品上下文一致。

#### 4.4.4 编辑元数据

- **入口**：更多菜单“编辑”。
- **可编辑项**：标题、简介、状态、标签、封面。
- **反馈**：成功后卡片即时更新；统计区同步变化；失败回退并提示重试。

#### 4.4.5 设置封面

- **入口**：更多菜单“设置封面”或封面区域动作。
- **约束**：封面 ≤ 400KB（与现有口径对齐）。
- **反馈**：上传成功后封面即时更新；失败提示原因（过大/格式/写回失败）。

#### 4.4.6 导出（单作品）

- **入口**：更多菜单“导出”。
- **导出内容（最终态）**：
  - 作品元数据
  - 章节正文
  - 作品相关的锦囊/概要/推演产物（若属于该作品的数据域）
- **格式**：ZIP / JSON / Markdown（最终以 v0）。

#### 4.4.7 归档 / 删除

- **入口**：更多菜单。
- **归档**：可撤销；默认筛选不显示归档作品。
- **删除**：高危二次确认；明确影响范围（章节、锦囊、推演产物、聊天关联等）与可恢复策略（若最终态提供回收站）。

### 4.5 批量操作（若 v0 目标态包含）

- **进入方式**：列表视图勾选/工具栏批量入口。
- **批量动作**：归档、导出、删除、打标签。
- **失败策略**：部分失败需逐项提示并可重试；整体回滚策略在“数据与写回总览”统一。

## 5. 数据与写回（读/写/缓存）

### 5.1 数据读取

- **作品列表**：作品标题、简介、状态、封面、标签、创建/更新时间、进度指针等。
- **统计区聚合**：
  - 作品总数
  - 有章节作品数
  - 总字数（万字）
  - 章节总数
- **最近编辑信息**：用于“最近编辑”排序与展示。

### 5.2 数据写回

- 创建：新建作品
- 更新：作品元数据、封面、标签、状态、归档状态
- 删除：作品及其关联数据（级联范围在数据总览统一）
- 导入：创建作品与章节，并写入导入来源元信息（若 v0 需要可追溯）

### 5.3 缓存与一致性（最终态要求）

- 写回成功后，本页必须保证“列表信息与统计区一致”。
- 若聚合统计异步更新，需提供明确刷新策略（自动刷新或提示刷新）。

## 6. 跨模块联动（因果链）

### 6.1 留白 → 写作

- **触发点**：进入作品（卡片点击/继续写作）。
- **落点表现**：进入 `/work/:id` 并定位上次进度；写作页加载作品/章节/锦囊/草稿等相关数据。
- **联动来源**：作品与章节数据 + `progressCursor`。

### 6.2 留白 → 推演

- **触发点**：卡片“推演”。
- **落点表现**：推演页预选作品；加载大纲/扫描/引用等与该作品相关的数据域。
- **联动来源**：作品上下文 + 推演数据 + 锦囊数据 +（可选）藏经引用。

### 6.3 留白 → 锦囊 / 概要

- **触发点**：卡片“锦囊”“概要”。
- **落点表现**：打开作品子页；子页读取同一作品域数据并保持一致。
- **联动来源**：`workId` 与其关联实体。

### 6.4 其它页 → 留白

- **触发点**：
  - 在 **写作** 页编辑导致字数/更新时间变化
  - 在锦囊/推演修改作品元信息或产生新章节
- **本页表现**：作品卡片字数/章节数/更新时间更新；统计区同步变化；筛选结果实时反映最新状态。

---

## 7. 空态 / 加载态 / 错误态

### 7.1 空态

- **无作品**：引导新建/导入（主入口明确）。
- **搜索无结果**：提示清空搜索/筛选并给一键清空。
- **筛选无结果**：提示调整筛选条件。

### 7.2 加载态

- 初次进入：Header/统计区/列表 skeleton（或“加载中…”）。
- 导入中：导入按钮 loading；必要时展示解析进度与可取消。

### 7.3 错误态与自救

- **读取失败**：提示“重试”；若涉及云端同步，提示检查网络与同步设置。
- **写回失败**：提示“未保存成功”，可重试；不可静默失败。
- **导入失败**：提示具体原因（格式/损坏/大小/解析），给替代方案与重试入口。

## 8. 验收与手测脚本（可执行）

### 8.1 基础路径（必过）

1. 进入 `/library`，确认 Header、统计卡、工具栏、列表显示正常。
2. 点击“新建作品”，仅填“作品名称”提交。
3. 创建成功后自动进入 **写作** 页；返回 `/library`，确认：
   - 作品卡出现
   - 作品总数统计 +1
4. 从该作品卡进入“推演”，确认推演页默认选中该作品。
5. 从该作品卡进入“锦囊”“概要”，确认进入对应子页且作品一致。

### 8.2 导入路径（必过）

1. 在 `/library` 点击“导入作品”，选择 `.txt` 或 `.docx`。
2. 导入完成后回到作品库，确认：
   - 新作品出现
   - 章节数/总字数统计更新
3. 进入 **写作** 页，确认章节存在且正文可读。

### 8.3 边界与错误（必过）

1. 搜索不存在关键字，确认“无结果”态与一键清空可用。
2. 对作品执行删除，确认二次确认清晰且删除后统计与列表一致。
3. 导入不可解析文件，确认提示清晰且不产生半成品作品。
