# 流光 — 灵感碎片管理（v0 目标态）

> 路由：`/inspiration` + 全局速记弹层  
> UI 口径：**最终 v0 对齐后的流光工作台**（多视图 + 文件夹/集合 + 多类型灵感卡 + 扩容与转入写作）。  
> 功能口径：**最终全部功能均已实现**（不写“待开发/占位”）。

---

## 1. 页面定位

- **一句话目的**：把“灵感/素材/引用/想法/书签/语音/图片”等碎片统一收纳、检索、扩容，并可靠推进到推演 / **写作页** / **锦囊** / 问策中形成闭环（「落笔」为 §5 总域名，正文编辑统一称 **写作**）。
- **主要用户任务**：
  - 快速捕获碎片（页内新建 + 全局速记）
  - 结构化管理（文件夹/集合/标签/收藏/私密）
  - 多视图浏览（网格/列表/瀑布流）与筛选检索
  - AI 五段扩容（生成候选扩写 → 保存为新碎片/转入正文）
  - 转入写作（三方式：章末追加 / 光标位插入 / 侧栏草稿合并）
  - 随机抽取与批量粘贴（提高再利用效率）

---

## 2. 入口与导航

### 2.1 进入本页

- **主入口**：Hub 顶栏“流光”。
- **深链进入（最终态要求）**：
  - 支持从 **写作页** / 推演跳入并自动带入“当前作品”作为筛选条件（便于就地找素材）。
  - 支持从全局速记保存后“一键去流光扩容”（直接打开流光并定位到新碎片、自动打开扩容面板）。

### 2.2 从本页离开（去哪里）

- **去写作页**：对任意碎片执行“转入写作”动作（进入 `/work/:id` 并定位插入点 / 草稿合并弹窗）。
- **去推演**：把碎片“关联到作品/人物/情节”后，在推演中作为节点材料引用（最终态双向跳转）。
- **去问策**：将碎片作为对话材料/引用，打开 `/chat` 并预填（最终态）。

### 2.3 状态保留（最终态要求）

- 记忆项：当前文件夹/集合、类型筛选、搜索词、视图模式（网格/列表/瀑布流）、排序、是否仅看收藏、是否包含私密。
- 返回行为：从 **写作页** 插入后返回流光，保持筛选 / 滚动位置与当前选中卡片（若 v0 要求保持）。

## 3. 信息架构（v0 目标 UI）

> v0 `liuguang-module.tsx` 的核心骨架是“左侧文件夹/集合 + 顶部工具条 + 多视图卡片区 + 右侧/弹层详情与扩容”。

### 3.1 顶部工具条（Header / Toolbar）

从左到右（最终态）：

- **页面标题**：流光 / 灵感库（低噪说明）
- **全局动作**：
  - 新建碎片（`+`）
  - AI 扩容（`Sparkles`）——对当前选中碎片或当前筛选集
  - 导入/导出（`Upload` / `Download`）——批量素材管理
- **搜索**：关键词搜索（标题/内容/来源/标签）
- **筛选**：类型、标签、收藏/私密、关联作品等
- **视图切换**：
  - 网格（`Grid3X3`）
  - 列表（`List`）
  - 瀑布流/密铺（最终态若保留）

显示条件：

- 当无碎片时，工具条仍显示（便于新建/导入/速记）。

### 3.2 左侧栏：文件夹 / 集合（Folders）

结构（从上到下）：

- 文件夹列表（可折叠）：
  - 颜色标识、名称、数量
  - 展开后可见子文件夹（若最终态支持层级）
- “未分类”作为兜底
- 文件夹管理入口：
  - 新建文件夹/重命名/删除/调整颜色

### 3.3 中间主栏：碎片列表（Cards）

#### 3.3.1 卡片通用结构（Grid / Masonry）

- 顶部类型条：类型图标 + 类型名；右侧显示收藏/私密/心情指示
- 标题（可选）
- 内容摘要（行数截断）
- 标签 chips
- 关联徽标：关联作品/人物/情节（以 badge 呈现）
- 悬浮操作（或卡片右上角菜单）：
  - 关联到作品（Link2）
  - 编辑（Pencil）
  - 更多菜单（`...`）：复制、收藏/取消收藏、归档、删除

#### 3.3.2 列表视图（List Row）

列表更强调“批量管理与密度”：

- 左：类型图标 + 标题/内容两行
- 中：标签/关联 badge
- 右：创建时间（相对时间）+ 快捷操作按钮（关联/编辑/菜单）

### 3.4 详情编辑与扩容（弹层 / 侧栏）

最终态以弹层为主（可实现为侧栏）：

- **碎片详情**：完整内容、标题、来源、URL、标签、关联作品/人物/情节、私密/收藏开关
- **编辑器**：支持富文本或纯文本（最终态与 **写作页** 一致的排版策略）
- **AI 五段扩容面板**：
  - 展示本次材料摘要与参数（最终态与全局 AI 口径一致）
  - 输出 5 段候选（可逐段收藏/复制/保存为新碎片/一键转入写作）

### 3.5 全局速记弹层（Global Capture）

这是“随时捕获”的核心入口：

- **唤起**：快捷键（默认 `Alt+S` / `⌥+S`，可在设置中改）
- **关闭**：`Esc`
- **保存**：`⌘/Ctrl+Enter`
- **字段（最终态）**：
  - 内容（必填）
  - 类型（文字/引用/想法等，默认文字）
  - 标签（可选）
  - 关联作品（可选，默认当前作品上下文）
  - 私密开关（可选）
- **保存后**：
  - 留在当前页面但提示“保存成功”
  - 或一键“去流光扩容”（跳转 `/inspiration` 并定位新碎片）

## 4. 功能清单（角落级）

> 本节按“原子动作”写清：入口、前置条件、过程、结果、失败提示与联动来源。

### 4.1 新建碎片（页内）

- **入口**：工具条 `+` / “新建碎片”按钮；或空态主按钮。
- **字段**：
  - 类型（text/quote/idea/bookmark/voice/image 等）
  - 标题（可选）
  - 内容（必填，语音/图片类型可同时有文字摘要）
  - 来源与链接（引用/书签类型）
  - 标签（多选 + 自定义）
  - 关联：作品/人物/情节（用于后续联动）
  - 收藏、私密
- **写回**：保存为 `InspirationFragment`（并写入 `updatedAt`）。
- **反馈**：新卡片出现在当前列表并高亮；统计数量更新（若 v0 有统计）。

### 4.2 全局速记（Alt+S）

- **入口**：`Alt+S` / `⌥+S`（可配置）。
- **前置条件**：无；在 Hub 与 **写作页** 都可唤起（但不得抢正文关键输入，最终态以“非侵入”优先）。
- **过程**：
  - 弹层出现，光标自动聚焦
  - 输入内容，选择类型/标签/关联作品（可选）
  - `⌘/Ctrl+Enter` 保存；`Esc` 取消
- **结果**：产生一条碎片，并在提示条中提供“去流光查看/扩容/转入写作”的快捷按钮。

### 4.3 视图切换（网格 / 列表 / 瀑布流）

- **入口**：工具条视图按钮。
- **规则**：切换不改变筛选/搜索；记忆上次选择。
- **反馈**：同一批碎片以不同密度呈现；卡片原子操作入口保持一致。

### 4.4 搜索与筛选

- **搜索范围（最终态）**：
  - 标题、内容、来源、URL、标签
  - 关联作品名（linkedWorks）
  - 关联人物/情节（若实现为结构化字段）
- **筛选**：
  - 类型（文字/语音/图片/引用/想法/书签）
  - 标签（多选）
  - 文件夹/集合
  - 收藏/私密
  - 关联作品
- **回显**：筛选条件以 chip/徽标展示，可一键清空。

### 4.5 卡片内操作

#### 4.5.1 复制内容

- **入口**：卡片 `...` 菜单“复制内容”。
- **结果**：写入剪贴板；toast 提示成功/失败。

#### 4.5.2 收藏 / 取消收藏

- **入口**：卡片 `...` 菜单；或卡片头部星标（若可点击）。
- **写回**：更新 `isFavorite`。
- **联动**：收藏筛选即时生效。

#### 4.5.3 私密 / 公开

- **入口**：详情编辑区的“私密”开关（最终态）。
- **写回**：更新 `isPrivate`。
- **规则**：私密碎片默认只在本机可见；若开启云端同步，私密策略与加密/访问控制在数据总览统一（最终态要求）。

#### 4.5.4 编辑碎片

- **入口**：卡片 Pencil；或双击卡片内容（最终态）。
- **可编辑**：标题/内容/类型/来源/URL/标签/关联/收藏/私密/文件夹归属。
- **写回**：更新 `updatedAt`；列表按“最近更新”排序会变化。

#### 4.5.5 删除 / 归档

- **入口**：卡片 `...` 菜单。
- **删除**：二次确认；若碎片被用于推演 / 问策 / **写作页** 等处引用，需要提示“删除后引用会失效 / 保留快照”（最终态策略）。
- **归档**：从默认视图隐藏，可在“归档筛选”中查看并可恢复（最终态）。

#### 4.5.6 关联到作品（Link）

- **入口**：卡片 Link2 按钮 / “关联到作品”动作。
- **行为**：选择作品（可多选）并写回 `workId` 或 `linkedWorks` 映射（以最终数据模型为准）。
- **联动**：
  - 在推演中可按作品筛选素材
  - 在 **写作页** AiPanel 中可选“引用流光素材”

### 4.6 AI 五段扩容（Inspiration Expand）

> 来自现有产品说明书：选中碎片 → AI 生成五段候选扩写；可保存为新碎片；可取消；有粗估/超阈值确认。

- **入口**：
  - 卡片操作“AI 扩容”
  - 或工具条“AI 扩容”（对选中碎片/当前筛选集）
- **前置条件**：至少一条碎片；并有模型配置可用（最终态与设置联动）。
- **过程**：
  - 展示“本次使用材料”摘要（包含：原碎片内容、关联作品/标签、可选锦囊/风格卡注入、预估 token）
  - 超阈值弹出数字确认（与全局成本策略一致）
  - 支持取消（Abort）
- **输出（UI）**：5 段候选扩写，每段提供动作：
  - 复制
  - 保存为新碎片（标签自动追加“扩容”）
  - 转入写作（见 4.7）
  - 加入问策（把该段作为对话材料）
- **写回**：
  - 扩写候选可暂存（会话级）或持久化为“扩容结果集合”（最终态）
  - 保存为新碎片写入 `InspirationFragment`

### 4.7 转入写作（三种方式）

> 来自现有产品说明书：章末追加 / 光标位插入 / 侧栏草稿合并。

- **入口**：碎片详情/扩容候选的“转入写作”动作。
- **前置条件**：需要目标作品与目标章节（若未选，弹出选择器；默认用当前作品上下文）。

三种方式：

- **章末追加**：
  - 进入 `/work/:id` 并定位到章节末尾
  - 以追加方式插入文本（保留分隔符与来源标记，最终态可配置）
- **光标位插入**：
  - 若已有打开的 **写作** 编辑器：直接插入到当前光标位
  - 若从流光跳转进入：进入 **写作页** 后提示“点击插入”并在插入时定位
- **侧栏草稿合并**：
  - 把内容写入 AiPanel 草稿区（或流光草稿区）并打开合并对话框（diff/插入/替换选区）

联动来源：

- 写入 `liubai:aiPanelDraft...`（与现有实现口径一致）
- 写作页的合并机制（AiDraftMergeDialog）负责落盘到正文

### 4.8 随机一条（Shuffle）

- **入口**：工具条“随机”按钮（`Shuffle`）。
- **行为**：从当前筛选结果中抽取 1 条并滚动定位、高亮若干秒。
- **用途**：在写作卡壳时快速唤起素材。

### 4.9 批量粘贴（Batch Paste）

- **入口**：工具条“批量粘贴”或空态入口（最终态）。
- **行为**：
  - 粘贴多段文本（按分隔符拆分为多条碎片）
  - 自动打标签/归入文件夹（可选）
- **写回**：批量创建 `InspirationFragment`；失败需逐条提示并允许重试。

### 4.10 导入 / 导出（素材级）

- **导入**：
  - 支持从文本/JSON/ZIP 导入碎片（最终态）
  - 导入冲突（ID/重复）策略：合并/跳过/重命名（在数据总览统一）
- **导出**：
  - 导出当前筛选集或全量碎片
  - 导出格式：JSON/Markdown/ZIP（最终态）

## 5. 数据与写回（读/写/缓存）

### 5.1 数据模型（与现有产品说明书对齐，并扩展 v0 结构）

- **InspirationFragment**（现有口径）：`id, workId?, collectionId?, body, tags[], createdAt, updatedAt`
- **InspirationCollection**（现有口径）：`id, name, sortOrder, createdAt, updatedAt`

v0 目标态扩展字段（最终态）：

- type（text/voice/image/quote/idea/bookmark）
- title、source、sourceUrl、audioUrl、imageUrl
- linkedWorks、linkedCharacters、linkedPlots
- isFavorite、isPrivate
- mood、location、category
- folderId（多层文件夹/颜色/排序）

### 5.2 读取

- 碎片列表：按筛选/搜索/文件夹/集合获取
- 关联作品列表：用于“关联到作品”选择器
- 标签建议：用于快速打标

### 5.3 写回

- 创建/更新/删除/归档碎片
- 文件夹/集合的创建/重命名/排序
- 扩容结果：会话缓存或持久化（最终态）

### 5.4 缓存与一致性（最终态要求）

- 扩容生成的中间结果必须“不丢输入”：刷新/跳转后仍可恢复（至少在当前会话）。
- 批量操作（导入/批量粘贴）必须提供“部分失败”可追溯与重试。

## 6. 跨模块联动（因果链）

### 6.1 流光 → 写作页

- **触发点**：转入写作（三方式）；或扩容候选“一键转入”。
- **落点表现**：进入 `/work/:id` 并插入/生成草稿/打开合并弹窗。
- **联动来源**：碎片内容 + 目标作品/章节 +（可选）标签/来源信息。

### 6.2 流光 → 推演

- **触发点**：关联到作品/人物/情节；或在推演中引用某碎片。
- **落点表现**：
  - 推演节点的“参考材料”出现该碎片引用
  - 可一键“应用到大纲/文策”（把碎片要点写入节点字段或生成文策记录）
- **来源**：碎片结构化字段（标签/关联）+ 推演节点映射。

### 6.3 流光 → 问策

- **触发点**：对碎片/扩容候选执行“去问策”。
- **落点表现**：打开 `/chat`，预填引用文本；绑定关联作品与标签侧写。
- **来源**：碎片内容 + 作品上下文 +（可选）锦囊索引。

### 6.4 其它页 → 流光

- **写作页 / 推演生成素材**：可一键“存为流光碎片”（把草稿 / 节点摘要保存为碎片）。
- **藏经提炼要点**：可一键“拆成多条碎片”进入流光库。

---

## 7. 空态 / 加载态 / 错误态

### 7.1 空态

- **无碎片**：展示引导 + 新建碎片 / 导入 / 全局速记说明。
- **筛选无结果**：提示清空筛选/搜索并提供一键清空。
- **无作品上下文**：转入写作时提示先选择作品/章节。

### 7.2 加载态

- 初次进入：列表 skeleton；左侧文件夹 skeleton；卡片占位。
- AI 扩容中：按钮 loading + 可取消；候选区显示生成中占位。

### 7.3 错误态与自救

- **读写失败**：提示重试；必要时提示检查同步/网络。
- **剪贴板失败**：提示浏览器权限与重试。
- **AI 错误**：
  - Key/模型不可用：提示打开设置
  - 上下文过长：提供精简重试
  - 成本阈值：数字确认

## 8. 验收与手测脚本（可执行）

### 8.1 基础路径（必过）

1. 进入 `/inspiration`，确认左侧文件夹、顶部工具条、碎片列表显示正常。
2. 新建一条文字碎片（带标签与关联作品），确认卡片出现且可搜索命中。
3. 切换网格/列表视图，确认信息等价且操作入口完整。

### 8.2 全局速记（必过）

1. 在任意 Hub 页面按 `Alt+S` / `⌥+S` 打开速记弹层。
2. 输入内容并 `⌘/Ctrl+Enter` 保存，确认提示成功。
3. 跳转到 `/inspiration`，确认新碎片存在并可定位。

### 8.3 扩容与转入写作（必过）

1. 选中某碎片执行“AI 五段扩容”，确认生成 5 段候选且可取消。
2. 将某候选“保存为新碎片”，确认标签追加“扩容”。
3. 将某候选“转入写作”分别测试三方式：
  - 章末追加
  - 光标位插入
  - 侧栏草稿合并（进入 **写作页** 能看到草稿并可合并入正文）

### 8.4 批量与随机（必过）

1. 批量粘贴多段文本，确认拆分成多条碎片且失败可追溯。
2. 点击“随机一条”，确认从当前筛选集抽取并滚动高亮。
