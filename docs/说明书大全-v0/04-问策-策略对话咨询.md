# 问策 — 策略对话咨询（v0 目标态）

> 路由：`/chat`  
> UI 口径：**最终 v0 对齐后的问策工作台**（会话列表 + 对话区 + 右侧策略/上下文/技法卡）。  
> 功能口径：**最终全部功能均已实现**（不写“待开发/占位”）。

---

## 1. 页面定位

- **一句话目的**：把“推演的分歧、写作的卡点、素材的选择、技法的应用”转为可追踪的对话与决策清单，并将结论可写回到推演 / **写作页**（草稿、正文）/ **锦囊**（设定条目）/ 流光。
- **主要用户任务**：
  - 新建/管理多会话（按作品维度组织）
  - 在对话中注入上下文（作品/章节/大纲节点/设定索引/引用素材）
  - 生成可执行的建议（行动项、改写方案、结构调整）
  - 把输出一键写回：推演文策/大纲字段、**写作**侧栏草稿、锦囊条目、流光碎片
  - 保存“技法卡”（可检索、可复用、可引用来源）

---

## 2. 入口与导航

### 2.1 进入本页

- **主入口**：Hub 顶栏“问策”。
- **从推演进入**：
  - 三分支预测“去问策跟进”
  - 扫描结果“转问策讨论”
  - 推演 AI 对话“一键形成议题”
  - 进入时可携带 `prefill`（预填问题）与 `preferWorkId`（优先作品）。
- **从流光进入**：对碎片/扩容候选“一键去问策”，以引用材料形式预填。
- **从写作页进入**：侧栏或选中段落“一键问策”，自动携带选区文本与章节上下文（最终态）。

### 2.2 从本页离开（去哪里）

- **去推演**：从会话关联的作品/节点跳转 `/logic` 并定位到对应节点或问题条目。
- **去写作页**：把回复“一键写入草稿 / 合并到正文”时跳转 `/work/:id` 并打开草稿区 / 合并弹窗。
- **去锦囊**：把建议“一键写入设定/术语/禁忌/时间线”时跳转 `/work/:id/bible` 并定位条目。
- **去流光**：将某段回复/引用“一键存为碎片”跳转 `/inspiration` 并定位新碎片。

### 2.3 状态保留（最终态要求）

- 记忆项：
  - 当前会话
  - 会话筛选（按作品/类型/置顶）
  - 右侧 Tab（上下文/模板/技法卡等）
  - 是否注入设定索引（`includeSettingIndex`）
  - 模型与参数（若问策有独立模型选择）
- 返回行为：从推演 / **写作页** 跳回问策时，会话不丢、滚动位置不跳、草稿输入不丢。

## 3. 信息架构（v0 目标 UI）

> v0 `wence-module.tsx` 的骨架是：左侧会话列表，中间对话主区，右侧策略辅助面板（Tabs）。

### 3.1 顶部条（Top Bar）

- **左侧**：
  - 左栏开关（收起/展开会话列表，窄屏必备）
  - 当前会话标题（可重命名）
  - 当前会话策略类型徽标（情节/人物/世界观/节奏/冲突/伏笔）
- **右侧**：
  - 模型选择器（若提供）
  - 刷新/重试（对最近一次失败请求）
  - 设置入口（问策相关设置）

### 3.2 左侧栏：会话列表（Conversations）

- **会话分组/筛选**：
  - 置顶会话
  - 最近会话（按更新时间）
  - 按关联作品筛选（最终态支持）
  - 搜索会话（标题/最后一条消息）
- **会话项展示**：
  - 标题
  - 策略类型图标 + 颜色
  - 最后一条消息摘要
  - 时间戳（相对时间）
  - 消息数
  - 关联作品名（badge）
- **会话项操作（`...`）**：
  - 置顶/取消置顶
  - 重命名
  - 复制会话链接/摘要（最终态）
  - 删除（高危确认）
- **新建会话**：底部或顶部 `+` 按钮

### 3.3 中间主区：对话（Chat）

从上到下：

- **系统消息条**：展示本次注入的上下文摘要（作品/章节/设定索引/引用材料等）。
- **消息列表**：
  - 用户消息（右侧气泡）
  - 助手消息（左侧气泡）
  - 每条消息包含：时间戳、复制、赞/踩（反馈）、引用/建议/行动项（如有）
- **生成中状态**：助手头像 + 三点动效
- **输入区**：
  - 多行输入（Textarea）
  - Enter 发送、Shift+Enter 换行
  - 发送按钮（禁用态/加载态）
  - 计数与提示（如 token 预估/成本阈值，最终态可显示在次要区域）

### 3.4 右侧辅助区（Tabs）

右侧是“把对话变成可执行”的关键：

- **Tab A：上下文**（最终态）
  - 关联作品选择（决定注入哪些风格卡/标签侧写）
  - 设定索引开关（`includeSettingIndex`：人物/世界观/术语/禁忌/时间线）
  - 引用素材列表：来自流光 / 藏经 / 推演节点 / **写作页**选区
  - 本次使用材料摘要（可折叠，和全局 AI 口径一致）
- **Tab B：模板**（快捷模板胶囊）
  - 预设问题按钮：分析人物动机/设计冲突升级/调整叙事节奏/伏笔回收/情感共鸣/章节目标检查…
  - 点击后填入输入框（不立即发送）
- **Tab C：技法卡**（Technique Cards）
  - 列表可检索
  - 新建技法卡：标题/摘要/标签/来源书名
  - 从对话中“一键保存为技法卡”
  - 技法卡可一键应用：生成下一条提问、或写入推演文策（最终态）

> v0 若包含更多 Tab（例如“参考/引用/行动项”），在最终态可扩展，但骨架不变。

## 4. 功能清单（角落级）

### 4.1 多会话管理

- **新建会话**：
  - 入口：左侧 `+` 或空态按钮
  - 选择策略类型：情节/人物/世界观/节奏/冲突/伏笔
  - 可选：关联作品（推荐）
  - 结果：会话出现在列表顶部并自动进入
- **切换会话**：点击会话项 → 中间对话区加载历史消息
- **重命名**：
  - 入口：会话标题点击编辑 或 `...` 菜单
  - 写回：更新会话 title
- **置顶/取消置顶**：
  - 入口：`...` 菜单或 pin 快捷按钮
  - 行为：置顶会话固定在列表顶部
- **删除会话**：
  - 入口：`...` 菜单“删除”
  - 高危确认：明确删除范围（消息、行动项、引用快照）
  - 写回：移除会话记录；若云端同步开启同时删除云端（最终态）

### 4.2 对话与生成

- **发送消息**：
  - 入口：输入框 Enter 或“发送”
  - 行为：追加用户消息 → 进入生成中 → 流式输出助手消息
  - 失败：显示错误气泡 + “重试/复制错误/打开设置”
- **停止生成**：
  - 入口：生成中显示“停止”按钮（最终态）
  - 行为：Abort 当前请求，保留已生成内容
- **复制消息**：
  - 入口：消息右侧复制按钮
  - 反馈：toast
- **赞/踩反馈**：
  - 入口：助手消息下方
  - 写回：本地记录用于改进提示词/模型偏好（最终态）

### 4.3 上下文注入（问策的核心）

> 目标：让问策的每次回答都“有根据、可复现、可写回”。

- **关联作品**：
  - 入口：右侧“上下文”Tab 的作品选择器
  - 行为：绑定 workId，决定后续默认注入：风格卡、标签侧写、章节摘要等
- **设定索引**（`includeSettingIndex`）：
  - 入口：右侧开关/复选
  - 内容：人物/世界观/术语/禁忌/时间线（按全局策略截断）
  - 行为：开启后，在系统消息条显示“已加载设定索引…”
- **引用材料**：
  - 来源：
    - 推演：三分支/扫描项/节点摘要
    - 流光：碎片/扩容候选
    - 藏经：提炼要点
    - **写作页**：选区 / 本章摘要 / 邻章摘要
  - UI：以 badge/列表形式呈现，可逐条移除

### 4.4 快捷模板（胶囊）

- **入口**：右侧“模板”Tab
- **行为**：点击胶囊 → 将 prompt 写入输入框
- **可编辑**：用户可自定义模板并排序（最终态）
- **联动**：推演可把某个分支/冲突点一键生成“模板提问”（最终态）

### 4.5 技法卡（Technique Cards）

- **从对话保存**：
  - 入口：助手消息上的“一键保存为技法卡”
  - 字段：标题/摘要/标签/来源书名（可编辑）
  - 写回：保存到技法卡库（本地或云端，最终态在数据总览说明）
- **列表检索与管理**：
  - 搜索：标题/标签/来源
  - 编辑/删除/置顶（若支持）
- **应用技法卡**：
  - 一键生成下一条提问（把技法卡作为上下文）
  - 一键写入推演文策（形成可追溯记录）

### 4.6 从问策“一键写回”到其它模块（最终态关键）

问策的每条输出都应支持明确的写回动作：

- **写回到写作页（AiPanel 草稿）**：
  - 动作：将某段建议写入 AiPanel 草稿区（支持合并/插入/替换）
  - 联动来源：`liubai:aiPanelDraft...` 或等价草稿存储
- **写回到推演（文策/节点字段）**：
  - 动作：
    - 保存为文策记录（决策/修订/AI建议）
    - 应用到大纲节点字段（摘要/冲突点/情绪曲线/人物地点关联）
  - 记录来源：必须写入引用（来自哪个会话/哪条消息）
- **写回到锦囊**：
  - 动作：把建议转成“设定条目/术语/禁忌/时间线事件”
  - 结果：在 `/work/:id/bible` 可定位到新增条目
- **写回到流光**：
  - 动作：把某段回复保存为碎片（类型：想法/引用/文字）
  - 标签：自动追加“问策”

## 5. 数据与写回（读/写/缓存）

### 5.1 数据模型（与现有产品说明书对齐）

- **WenceChatSession（云）**：`id, userId, workId?, title, includeSettingIndex, messages(jsonb), updatedAt`
- **本地存储**：`localStorage`（按 key 持久化，会话与技法卡分开）
- **技法卡**：`liubai:techniqueCards`（现有口径）

### 5.2 读取

- 会话列表（按更新时间、置顶、作品筛选）
- 当前会话消息（流式输出 + 历史）
- 作品上下文（风格卡、设定索引）
- 引用材料（流光 / 推演 / 藏经 / **写作页**）

### 5.3 写回

- 会话元信息：标题、置顶、关联作品、includeSettingIndex
- 会话消息：messages 追加、更新、删除（最终态支持删除单条/清空会话）
- 技法卡：新增/编辑/删除/检索索引
- 写回到其它模块的产物（草稿/文策/锦囊/碎片）按对应模块数据域落盘

### 5.4 缓存与一致性（最终态要求）

- 网络中断或刷新：输入框草稿不丢；生成中的结果可恢复（至少保留到会话）。
- 云端开启时：跨设备同步以 `updatedAt` 为准；冲突时提供“保留两份/合并”策略。

## 6. 跨模块联动（因果链）

### 6.1 推演 → 问策

- **触发点**：三分支“去问策跟进”、扫描结果“转问策”、节点冲突点“问策讨论”。
- **落点表现**：打开 `/chat` 新会话，预填问题，并自动关联作品与选中节点。
- **来源**：推演节点/扫描项/分支内容（作为引用材料）。

### 6.2 流光 → 问策

- **触发点**：碎片“去问策”、扩容候选“问策评估”。
- **落点表现**：预填引用 + 自动加标签（剧情/人物/世界观等）并可选择策略类型。
- **来源**：碎片内容与结构化标签。

### 6.3 写作页 → 问策

- **触发点**：选区问策 / 章节卡点问策。
- **落点表现**：预填选区 + 注入本章摘要 / 锦囊索引；输出可直接写入 **写作页** 草稿。

### 6.4 问策 → 推演 / 写作页 / 锦囊 / 流光

- **推演**：写入文策或应用到节点字段
- **写作页**：写入 AiPanel 草稿并合并入正文
- **锦囊**：新增设定条目/术语/禁忌/时间线
- **流光**：保存为碎片并可继续扩容

---

## 7. 空态 / 加载态 / 错误态

### 7.1 空态

- **无会话**：引导新建会话，并展示快捷模板示例。
- **无消息**：显示“从一个问题开始”，并推荐模板胶囊。
- **无作品上下文**：仍可对话，但提示“关联作品可获得更准建议”。

### 7.2 加载态

- 切换会话：消息区 skeleton；右侧上下文同步刷新。
- 生成中：显示生成气泡与停止按钮；发送按钮禁用。

### 7.3 错误态与自救

- **模型配置错误/不可用**：提示“打开设置”。
- **网络失败**：提示“重试/复制输入/保存草稿”。
- **上下文过长**：提示“精简并重试”，并展示会被裁剪的来源列表。

## 8. 验收与手测脚本（可执行）

### 8.1 会话管理（必过）

1. 进入 `/chat`，新建会话并选择策略类型。
2. 重命名会话、置顶会话、删除会话（含确认）均可用。
3. 按关联作品筛选会话可用，搜索会话标题可用。

### 8.2 上下文注入（必过）

1. 关联作品并开启设定索引，确认系统消息显示“已加载设定…”
2. 从推演三分支“去问策跟进”进入问策，确认预填与引用材料存在。
3. 从流光碎片“去问策”进入，确认引用材料存在且可移除。

### 8.3 对话与写回（必过）

1. 发送消息，确认流式输出与停止按钮可用。
2. 将某段回复“写入写作侧栏草稿”，进入 `/work/:id` 可看到草稿并可合并。
3. 将某段回复“写入推演文策”，进入 `/logic` 文策视图可看到记录并可追溯来源。
4. 将某段回复“写入锦囊条目”，进入 `/work/:id/bible` 可定位新增条目。
5. 将某段回复“存为流光碎片”，进入 `/inspiration` 可定位新碎片。
