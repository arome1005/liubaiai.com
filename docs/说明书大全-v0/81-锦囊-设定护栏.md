# 锦囊 — 设定护栏（v0 目标态）

> **本页 = 「锦囊」**（原「圣经」设定护栏）；**≠** 把「落笔」整个模块等同于本页——**「落笔」** 仍是七模块名与 §5 总域（提示词库 / 模板 / 笔感 / 词典等亦属落笔）。顶栏「落笔」当前实现常进入 `**/work/:id/bible`**，与本文同页。「圣经」为技术/行业别名，UI 称「锦囊」。见 `design/seven-modules-ui-spec.md` §5。  
> 路由：`/work/:id/bible`  
> UI 口径：以 v0 最终“作品子导航（写作 / 概要 / 锦囊 / 作品库）+ 锦囊多分区编辑台”呈现。  
> 功能口径：**最终全部功能均已实现**（不写“待开发/占位”）。  
> **曾误写（已纠 2026-04-10）**：勿将「落笔」指代为 `/work/:id` 全屏正文页；正文编辑为 **「写作」**，见 `05-写作-本书正文编辑.md`。

---

## 1. 页面定位

- **一句话目的**：把作品的“不可违背的事实与风格约束”结构化沉淀为可检索、可注入、可扫描的设定护栏，支撑推演扫描、生辉仿写与**写作**侧栏 AI 注入。
- **主要用户任务**：
  - 建立并维护全书设定：人物/世界观/时间线/术语/伏笔/风格卡
  - 维护“禁忌与禁写”：避免写作与 AI 输出踩雷
  - 提供“设定索引”：用于问策与**写作**侧栏 AI 的上下文装配
  - 支撑推演一致性扫描：成为“事实基线”
  - 支持模板与笔感：让生成更像作者本人

---

## 2. 入口与导航

### 2.1 进入本页

- **从写作页进入**：
  - 写作右侧“锦囊”Tab → “打开锦囊”
  - 写作/AI 提示中的“查看设定”跳转
- **从顶栏「落笔」进入**：当前应对齐 `/work/:id/bible`（落笔域内锦囊子页；若代码仍链他处，以 spec 为准待对齐）
- **从推演进入**：
  - 扫描结果点击“打开锦囊对应条目”
  - 时间线快速追加后跳转确认
- **从作品库进入**：作品卡片快捷入口“锦囊”

### 2.2 从本页离开（去哪里）

- 返回**写作**：继续编辑正文（保持章节与光标）
- 去推演：从设定条目“一键推演扫描/加入冲突点”
- 去概要：查看摘要覆盖范围与更新时间
- 去作品库：回到留白

### 2.3 状态保留（最终态要求）

- 记忆项：
  - 当前分区（人物/世界观/伏笔/时间线/术语/风格卡/模板/笔感）
  - 搜索词与筛选条件
  - 当前选中条目与滚动位置
- 返回行为：从推演/**写作**跳回锦囊时仍定位原条目（尤其是由扫描结果跳入的定位）。

## 3. 信息架构（v0 目标 UI）

> 锦囊页是“分区导航 + 条目列表 + 条目编辑/详情”的三段结构，配合作品子导航。

### 3.1 作品子导航（Work Subnav）

顶栏下方固定出现：

- 写作 `/work/:id`
- 概要 `/work/:id/summary`
- 锦囊 `/work/:id/bible`（当前高亮）
- 作品库 `/library`

### 3.2 分区导航（Sections）

分区（与现有产品说明书口径一致）：

- **人物**
- **世界观**
- **伏笔**
- **时间线**
- **术语表**
- **风格卡**
- **提示词模板**
- **笔感样本**

### 3.3 条目列表（List）

共性能力（最终态）：

- 搜索：标题/内容/标签
- 筛选：优先级、状态、是否置顶/收藏、关联章节
- 排序：更新时间/优先级/名称
- 批量操作：批量改状态/批量打标签/批量删除（若 v0 有）

### 3.4 条目详情/编辑（Editor）

共性字段（最终态建议）：

- 标题
- 内容（多行，支持 markdown 或结构化段落）
- 标签
- 优先级（高/中/低）
- 状态（active/retired/draft）
- 关联章节（可多选）
- 置顶（pinned）

并提供统一的：

- 保存/撤销
- 删除（高危确认）
- 复制内容
- “写入 AI 注入材料”预览（显示会如何被装配）

## 4. 功能清单（角落级）

### 4.1 人物（BibleCharacter）

> 功能口径（现有说明书）：名称、动机、关系、语气笔记、禁忌；排序。

- **新建人物**：填写最小字段（名称/定位）→ 保存
- **关系与动机**：结构化编辑（最终态可拆字段，便于检索与注入）
- **语气笔记**：用于对话生成的一致性提示
- **禁忌**：人物禁忌会进入推演扫描与写作侧栏 AI 禁写装配
- **联动**：
  - 推演规则扫描：人物禁忌命中告警
  - 问策设定索引：人物列表可勾选注入
  - **写作**/生辉：人物条目摘要进入“本次使用材料”

### 4.2 世界观（BibleWorldEntry）

- **按类型组织**：势力/规则/地理/力量体系等
- **条目编辑**：支持长文本、要点列表、引用来源
- **联动**：同人物，作为设定索引与扫描基线

### 4.3 伏笔（BibleForeshadow）

> 口径：标题、埋设位置、预计回收、状态（待定/已解/弃用）、备注、关联章节。

- **字段**：
  - 标题
  - 埋设位置（章节/场景/正文片段引用）
  - 预计回收章节
  - 状态：待定/已解/弃用
  - 备注
- **联动**：
  - 推演：伏笔作为冲突点/规划节点参考
  - **写作**：写作时提醒“此处需回收伏笔”
  - 概要：章节概要可标记“伏笔推进/回收”

### 4.4 时间线（BibleTimelineEvent）

> 口径：事件标签、备注、关联章节；推演可快速追加。

- **新增事件**：
  - 入口：时间线分区“新增”
  - 字段：事件名、标签、备注、关联章节、时间点（可选）
- **排序与筛选**：按时间/章节/标签
- **联动**：
  - 推演快速追加写回本分区
  - 规则/AI 扫描引用时间线作为事实基线

### 4.5 术语表（BibleGlossaryTerm）

> 口径：词条、类别（人名/术语/已死）、备注。

- **类别**：
  - 人名（可映射到人物）
  - 术语（设定名词）
  - 已死（用于禁写/逻辑一致性）
- **联动**：
  - 推演扫描：术语一致性
  - 写作侧栏 AI：术语表注入与禁用提示
  - 问策：设定索引勾选注入

### 4.6 风格卡（WorkStyleCard）

> 口径：人称/视角、调性、禁用套话、文风锚点、额外约束。每作品一份。

- **内容结构（最终态）**：
  - POV / 视角限制
  - 调性与节奏
  - 禁用套话（可作为列表）
  - 文风锚点（示例句/段落）
  - 额外约束（例如“少用感叹号”）
- **联动**：
  - 推演：扫描与三分支注入
  - 写作 AiPanel：默认注入并在“本次使用材料”展示
  - 生辉：仿写强约束

### 4.7 提示词模板（WritingPromptTemplate）

> 口径：可复用片段，写作侧栏“额外要求”一键装配。

- **管理**：新建/编辑/删除/分类
- **使用**：在写作 AiPanel 一键插入到“额外要求”
- **联动**：问策也可引用模板快速提问（最终态）

### 4.8 笔感样本（WritingStyleSample）

> 口径：参考段落，注入写作侧栏 user 上下文。

- **来源**：
  - 从正文选区“存为笔感样本”
  - 从外部粘贴
- **用途**：增强仿写/润色的“像作者本人”
- **联动**：
  - 写作 AiPanel 勾选注入
  - 生辉上下文项（style sample）

## 5. 数据与写回（读/写/缓存）

### 5.1 数据模型（与现有说明书对齐）

- `BibleCharacter`
- `BibleWorldEntry`
- `BibleForeshadow`
- `BibleTimelineEvent`
- `BibleGlossaryTerm`
- `WorkStyleCard`（每作品一份）
- `BibleChapterTemplate`（章头/章尾模板）
- `WritingPromptTemplate`
- `WritingStyleSample`
- 章节级锦囊字段：`ChapterBible`（目标/禁写/POV/场景/人物状态）

### 5.2 写回原则（最终态）

- 所有条目写回必须可追溯（更新时间/来源）
- 删除为高危操作，必须二次确认
- 写回失败必须可重试且不丢输入

---

## 6. 跨模块联动（因果链）

- **锦囊 → 推演**：规则扫描与 AI 扫描注入禁写/禁忌/术语/人物/时间线。
- **锦囊 → 写作侧栏**：本章锦囊字段 + 全书锦囊板块 + 术语 + 笔感 → 装配器注入。
- **锦囊 → 生辉**：仿写注入风格卡 + 标签侧写 + 设定索引。
- **推演 → 锦囊**：时间线快速追加写回；扫描结果可定位到条目。
- **问策 → 锦囊**：对话输出可一键写入设定条目/术语/禁忌/时间线（最终态）。

---

## 7. 空态 / 加载态 / 错误态

- **空态**：无条目时提供“从模板创建/从正文提取/从推演导入”的引导。
- **加载态**：条目列表 skeleton；保存按钮 loading。
- **错误态**：
  - 写回失败：提示重试
  - 冲突：提示重新载入/合并（若多端同步）

---

## 8. 验收与手测脚本（可执行）

1. 新建人物/世界观/术语条目并保存，刷新后仍在。
2. 在写作 AiPanel 勾选“锦囊/术语/笔感”，确认“本次使用材料”摘要正确。
3. 在推演运行规则扫描，确认会引用术语/禁忌并可跳转回锦囊条目。
4. 从推演快速追加时间线事件，回到锦囊时间线分区可见。
5. 从正文选区存为笔感样本，生辉/**写作**可勾选注入并生效。
