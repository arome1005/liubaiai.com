# 设置 — 系统配置中心（v0 目标态）

> 路由：`/settings`  
> UI 口径：**最终 v0 对齐后的设置中心**（左侧分类导航 + 右侧分区表单/面板）。  
> 功能口径：**最终全部功能均已实现**（不写“待开发/占位”）。

---

## 1. 页面定位

- **一句话目的**：把“账号、AI 模型、成本、编辑器偏好、快捷键、备份与数据管理”等全局能力集中配置，并为其它页面提供稳定的默认行为与安全门控。
- **主要用户任务**：
  - 配置各厂商模型与连通测试（Key/Base URL/模型）
  - 设置默认模型与各模块模型偏好（推演 / 流光 / 问策 / **落笔** / 生辉；其中「落笔」项覆盖 **写作页 AiPanel** 与依赖同套模型门控的生成链路，具体分组以 v0 UI 为准）
  - 配置成本阈值与确认策略、会话预算
  - 配置编辑器排版与沉浸/速记快捷键
  - 执行数据导出 ZIP、合并导入、全量恢复（危险）
  - 查看用量与预算预警（tokens/费用/趋势）

---

## 2. 入口与导航

### 2.1 进入本页

- **主入口**：Hub 顶栏设置图标；或任一页面错误提示中的“打开设置”链接（例如模型不可用/Key 缺失/网络类错误）。

### 2.2 从本页离开（去哪里）

- 返回上个页面（浏览器 Back）
- 跳转到：**写作页** / 锦囊页 / 推演 / 生辉（用于快速验证模型与编辑器偏好是否生效）

### 2.3 状态保留（最终态要求）

- 左侧“当前分类”可通过 URL hash/query 记忆（例如 `?tab=ai`），从错误跳入时直接定位到相关分区。
- 表单输入在未保存前刷新不丢（草稿缓存），或至少明确提示“未保存”。

## 3. 信息架构（v0 目标 UI）

> v0 `settings-module.tsx` 的骨架是“左侧分类列表 + 右侧对应设置面板”，并包含多个子组件：费用与额度、AI 设置、编辑器、外观、快捷键、数据管理、隐私等。

### 3.1 左侧分类导航（Categories）

常见分类（最终态以 v0 为准）：
- 账户
- 费用与额度
- AI 设置
- 编辑器
- 外观
- 快捷键
- 通知
- 数据管理
- 隐私安全

交互：
- 点击分类切换右侧面板
- 当前分类高亮
- 可在窄屏收起/展开

### 3.2 右侧内容区（Panels）

每个分类为一个可滚动面板，内部由卡片/Accordion 组织：
- 标题 + 一句话说明
- 表单区（输入/选择/开关/滑块）
- 危险区（红色边框/明确不可逆）
- 右上角“保存/应用”策略（即时保存或显式保存，最终态需统一）

## 4. 功能清单（角落级）

### 4.1 AI 厂商与模型配置（后端模型配置）

> 来自现有产品说明书口径：弹窗逐厂商 API Key、Base URL、模型选择、连通测试、模型健康度。

- **入口**：AI 设置分区中的“后端模型配置”按钮（打开 `BackendModelConfigModal` 等价弹窗）。
- **厂商维度**（最终态）：
  - OpenAI / Anthropic / Gemini / 豆包 / 智谱 / Kimi / 小米 / Ollama…
- **每个厂商配置项**：
  - API Key（加密/遮罩显示，支持一键清空）
  - Base URL（可选）
  - 默认模型（下拉）
  - 连接测试（成功/失败原因）
  - 健康度（最近一次可用性）
- **写回**：
  - Key 存储策略：本地安全存储（最终态需明确是否进入云端；推荐仅本地）
  - 变更生效范围：**写作页** AiPanel / 生辉 / 推演 AI 扫描 / 流光扩容 / 问策对话
- **失败与自救**：
  - Key 无效/权限不足：提示检查 Key 权限与计费
  - Base URL 不可达：提示网络/代理

### 4.2 默认与上下文（模型偏好与预算）

> 来自现有产品说明书口径：默认提供方、token 阈值、确认策略、侧栏会话预算。

- **默认模型**：全局默认（当模块未单独设置时使用）
- **按模块默认**：
  - 推演 / 流光 / 问策 / **落笔** / 生辉（「落笔」对应 §5 域内默认模型；正文侧主要作用于 **写作页** AiPanel）
- **token 阈值与确认策略**：
  - 粗估 token
  - 超阈值数字确认（阈值可配）
  - 冷却与最短停留（防误点）
- **会话预算**：
  - 本会话累计 tokens/费用
  - 本机累计 tokens/费用
  - 达到预算阈值时提示或阻断（最终态策略）

### 4.3 费用与额度（Billing / Usage）

- **本月用量总览**：
  - tokens、费用、预算、剩余天数
  - 预算进度条（Progress）
- **按模块统计**：推演 / 生辉 / 问策 / 落笔 / 流光占比（「落笔」统计含 **写作页** 与锦囊相关调用口径，以 v0 实现为准）
- **趋势图**（最终态）：每日 tokens/费用折线
- **预算预警**：
  - warning/critical 两档
  - 达到阈值提示（可配置阈值）
- **写回**：预算与预警策略保存到用户偏好

### 4.4 编辑器设置（写作页排版与沉浸；属正文编辑体验）

> 来自现有产品说明书口径：字号、行高、字体族、稿纸底色、快捷键配置。

- **排版**：
  - 字体族
  - 字号（例如 12–28）
  - 行高
  - 稿纸底色（paper tint）
- **编辑器宽度**：
  - 自定义宽度范围与默认值
  - 自适应开关
- **保存策略**：
  - 自动保存开关/间隔（最终态）
  - 冲突提示策略（乐观锁）
- **沉浸模式**：
  - `Alt+Z` 热键开关/改键
  - 进入沉浸时隐藏哪些区域（章节树/右侧栏）

### 4.5 快捷键设置（Hotkeys）

- **全局速记**：
  - 默认 `Alt+S` / `⌥+S`
  - 自定义组合键
  - 行为：Esc 关闭、⌘/Ctrl+Enter 保存
- **沉浸写作**：`Alt+Z`
- **命令面板**：`⌘/Ctrl+K`
- **冲突检测**：
  - 当用户设置冲突按键时，提示冲突来源与建议
- **写回**：保存到 `liubai:hotkeyConfig`（现有口径）

### 4.6 外观（Appearance）

- **主题**：
  - 浅色/深色/跟随系统
- **密度与可读性（最终态）**：
  - UI 密度（紧凑/舒适）
  - 对比度/边框清晰度（避免深色糊边）
- **写回**：本地偏好（并即时生效）

### 4.7 数据管理（备份与恢复）

> 来自现有产品说明书口径：导出 ZIP、全量恢复（危险）、合并导入、备份软提醒开关。

#### 4.7.1 导出 ZIP（全量备份）

- **入口**：数据管理 → “导出备份”
- **内容**：`data.json`（全量写作数据）+ 藏经数据（仍在本地）
- **反馈**：导出成功提示；失败提示原因（空间/权限）

#### 4.7.2 合并导入（Merge Import）

- **入口**：数据管理 → “合并导入”
- **规则**：
  - ID 映射后追加，不覆盖已有
  - 冲突策略：重命名/跳过（最终态）
- **反馈**：导入报告（新增/跳过/失败条目数）

#### 4.7.3 全量恢复（危险）

- **入口**：数据管理 → “全量恢复”
- **确认**：
  - 二次确认 + 明确不可逆范围（替换当前用户全部数据）
  - 可要求输入特定确认文案（最终态）
- **恢复后**：
  - 重启应用或自动刷新索引/缓存
  - 生成一条“恢复事件”可追溯（最终态）

#### 4.7.4 备份软提醒

- **开关**：约 30 天未导出时提示条（可关闭/可延后）

### 4.8 隐私与安全（在不写登录的前提下）

- **本地/云端边界提示**：
  - 写作数据本地优先、云同步可选
  - 藏经永不上云
- **日志与调试**（最终态）：
  - 是否记录调用日志
  - 是否脱敏（避免正文内容出现在日志）
- **危险操作统一策略**：删除/覆盖/大额调用需要二次确认

## 5. 数据与写回（读/写/缓存）

- **本地偏好**：
  - 主题、编辑器排版、快捷键、检索模式、预算阈值
- **模型配置**：
  - 厂商 Key/BaseURL/模型列表缓存
  - 连通测试结果与健康度
- **备份与导入**：
  - 导出 ZIP、导入映射表、恢复事件记录（最终态）

---

## 6. 跨模块联动（因果链）

- **设置 → 全模块**：模型与预算影响推演 / 流光 / 问策 / **落笔** / 生辉的默认模型与门控策略（「落笔」项与 §5 域、**写作页** AiPanel 等相关）。
- **设置 → 编辑器**：字号 / 行高 / 稿纸色 / 沉浸等 **即时作用于写作页** `/work/:id`。
- **设置 → 流光**：全局速记快捷键配置影响捕获弹层。
- **设置 → 藏经**：数据导出/恢复会影响藏经数据一致性与索引（恢复后需重建索引）。

---

## 7. 空态 / 加载态 / 错误态

- **模型未配置**：在 AI 设置中提示“未配置 Key”，并在其它页面的错误提示中提供直达。
- **连通测试失败**：显示具体错误（网络/认证/额度），并提供建议。
- **导入/恢复失败**：展示失败报告与重试入口；避免半成品状态。

## 8. 验收与手测脚本（可执行）

### 8.1 模型配置（必过）

1. 配置任一厂商 Key，执行连通测试成功。
2. 设置「落笔」默认模型为 A、生辉默认模型为 B，进入 **写作页** 与生辉确认调用模型一致。

### 8.2 成本与预算（必过）

1. 设置 token 阈值为较小值，触发一次高 token 操作，确认出现数字确认。
2. 设置预算预警阈值，确认达到阈值提示出现。

### 8.3 编辑器与快捷键（必过）

1. 修改字号 / 行高 / 稿纸底色，进入 **写作页** 确认即时生效。
2. 修改全局速记快捷键，验证新快捷键可唤起弹层。

### 8.4 备份与恢复（必过）

1. 导出 ZIP 成功。
2. 合并导入 ZIP，确认不覆盖且有导入报告。
3. 全量恢复触发二次确认；执行后数据被替换且应用正常可用。
