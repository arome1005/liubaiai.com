# 全局壳层与导航（v0 目标态）

> **术语（2026-04-11）**：**「落笔」** = 七模块名与 **§5 能力总域**。**「锦囊」** = 原「圣经」设定护栏的 UI 名，路由多为 **`/work/:id/bible`**，是落笔域内的 **子页**，与「落笔」模块名 **不互相替代**。**「写作」** = 全屏正文 **`/work/:id`**（见 spec §9），为新建作品后的 **默认主路径**。

## 1. 页面定位

- **覆盖范围**：除**写作**页 `/work/:id` 外的 Hub 页面（留白/推演/流光/问策/生辉/藏经/设置/子页等）
- **目标**：提供统一导航、作品上下文提示、全局命令面板、用户与设置入口，减少迷路与上下文丢失

## 2. 顶栏（Hub 顶栏）

- **品牌区**：返回主入口（默认 `/library`）
- **七模块导航**：留白 / 推演 / 流光 / 问策 / **落笔** / 生辉 / 藏经
  - **激活态**：高亮 + 底部短条
  - **「落笔」**：点击进入 **`/luobi`** 创作工具箱（**不依赖** 是否已有当前作品）  
  - **模块专篇**：七模块之一「落笔」的能力域、入口与分工见 **`08-落笔-能力域与入口.md`**
- **作品上下文区**：显示当前作品/章节/字数/标签，并提供快速跳转（继续写作/打开推演/切换作品）
- **全局搜索/命令面板入口**：`⌘/Ctrl+K`
- **用户菜单**：头像/邮箱、登出入口（本大全不展开登录/鉴权）
- **设置入口**：跳转 `/settings`

## 3. 命令面板（全局）

- **入口**：`⌘/Ctrl+K` 或顶栏按钮
- **能力**：跨模块搜索与跳转（作品/章节/标签/卡片/参考条目）
- **键盘行为**：上下选择、Enter 跳转、Esc 关闭

## 4. EditorShell（写作壳）

- **适用路由**：`/work/:id` 及其编辑相关的内部面板
- **差异**：不显示 Hub 七模块导航；提供写作专注与编辑器工具条

## 5. 通用 UI 约定

- **按钮状态**：默认/hover/active/disabled/loading
- **提示与确认**：高危操作二次确认（覆盖/删除/批量写回/大额 AI 调用）
- **加载**：骨架屏优先；不可避免的阻塞加载需给出明确文案与可取消
