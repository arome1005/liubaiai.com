## 终验收（建议按这个 checklist 跑）

> 目标：用最少步骤覆盖“七模块闭环 + 四条核心联动链路”，同时验证 v0 目标 IA 入口齐全。

### A. 基础导航与上下文

- 顶栏七模块导航齐全、激活态正确；从任意模块能返回 `/library`
- 选择作品后，全局作品上下文在 Hub 顶栏正确显示并影响其它模块默认选中
- `⌘/Ctrl+K` 命令面板可打开、可跳转（若最终态包含）

### B. 七模块基础能力（每个模块最小可用）

- 留白：新建作品/导入作品/搜索筛选排序/作品卡快捷入口可用
- 推演：左大纲树/中三视图/右三 Tab；可执行规则扫描、AI 扫描、三分支；时间线/地点可写回
- 流光：全局速记（Alt+S）、碎片管理、AI 五段扩容、三方式转入**写作**可用
- 问策：多会话、上下文注入（设定索引/引用材料）、模板/技法卡、一键写回可用
- **落笔（08 + 81）**：`08` 核对模块入口与能力域；`81` 验收锦囊子页分区编辑、保存，与生辉/**写作**侧栏注入联动（勿称「锦囊 = 落笔」）
- **本书编辑（写作）**：`/work/:id` 卷章管理、正文编辑自动保存、草稿恢复、快照、导出、AiPanel 草稿合并可用
- 生辉：上下文装配、多版本草稿、对比、采纳、写回**写作**草稿可用
- 藏经：导入书籍、分块索引、提炼、批量导出、RAG 命中进入装配器可用（且不上云）

### C. 四条核心联动链路（终验收必测）

- 链路 1：推演 → **写作**（写入草稿 → 合并入正文）
- 链路 2：推演 → 问策（分支跟进预填 → 输出写回推演 / **写作**草稿 / **锦囊**）
- 链路 3：流光 → **写作**（速记 → 扩容 → 三方式转入）
- 链路 4：藏经 → **写作** AI（RAG top-k → 本次使用材料 → 草稿合并）

### D. 数据安全与高危操作

- 删除/覆盖/全量恢复均有二次确认且明确影响范围
- AI 超阈值调用触发数字确认；错误提示能直达 `/settings`
- 写回失败不丢输入；冲突时可重新载入/合并

## 写作规范（本目录统一口径）

- **UI 口径**：以 v0 最终对齐后的布局、层级、密度、交互骨架为准。
- **功能口径**：以“最终全部功能已实现”为准；不写“待开发/占位”。
- **描述粒度**：细致到页面每个角落（区块/按钮/菜单/快捷键/空态/错误态/加载态）。
- **联动写法**：每个页面都必须写清：
  - **联动触发点**：本页哪些动作会影响其它页
  - **联动落点**：其它页会如何体现变化
  - **联动来源**：联动依赖的数据/能力来自哪个模块或哪类数据
