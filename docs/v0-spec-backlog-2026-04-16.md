## 2. 四条核心联动链路（拆解为可执行待办）

> 来源：`docs/说明书大全-v0/90-跨模块联动总览.md`。

### 链路 1：推演 → 写作（写入草稿 → 合并入正文）

- **P0**：推演侧“写入草稿”能携带来源信息（分支/章节/时间），写作 AiPanel 可见并可合并
- **P1**：合并后生成追溯记录（至少可回滚/快照）

### 链路 2：推演 → 问策（分支跟进预填）

- **P0**：从推演到问策可预填问题与引用材料可见
- **P0**：问策输出具备“一键写回到写作草稿/锦囊/推演文策”

### 链路 3：流光 → 写作（捕获/扩容 → 三方式转入）

- **P1**：三种转入方式均可用，且插入位置/草稿合并正确
- **P1**：转入后可回流流光并保持状态（返回不丢筛选/滚动）

### 链路 4：藏经 → 写作 AI（RAG）

- **P0**：写作 AiPanel 的 RAG 命中来源可追溯（书名/章节/片段）
- **P0**：藏经“提炼结构化”可作为 RAG/引用材料进入装配器
