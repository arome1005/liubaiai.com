## 怎么使用这份清单

### 重构前（5 分钟准备）

1. 打开编辑页，对着 B1–B7 每个弹窗截一张图，保存到桌面或任意位置备用。
2. 记住当前某一章的内容和标题（或截图），用于 A6 刷新对照。

### 每完成一个重构阶段后

1. 查看上面「优先顺序表」对应阶段的**自测重点列**。
2. 先跑 **A（冒烟）**——若 A 不过，停下来排查再继续。
3. 再跑对应的 **B / C / D / E / F** 子项。
4. 若有弹窗项，和之前的截图对比位置/大小是否一致。
5. **全部通过** → 该阶段完成，可进入下一阶段。
  **有不一致** → 回退该阶段改动，排查后重新提交。

### 判定标准

- **通过**：冒烟全绿 + 对应弹窗目测位置/大小/动线与截图一致。
- **不通过**：任何一项与重构前不一致 → 修复或回退后重测。
