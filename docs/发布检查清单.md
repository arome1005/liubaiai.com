## 发布检查清单（Web / Vercel）

最后更新：2026-04-02

### 0. 基本

- `npm run build` 通过
- **生产环境变量**：`VITE_SUPABASE_URL`、`VITE_SUPABASE_ANON_KEY` 已在托管平台配置并 **重新部署**；`/api` 已反代到 Node 或已设 `VITE_API_BASE`（见 `docs/生产环境部署.md`）
- **Supabase Auth**：Dashboard → URL Configuration 中 **Site URL**、**Redirect URLs** 与生产域名及本应用路由一致（见 `docs/生产环境部署.md` **Supabase Auth** 节；对应总体规划步 7）
- 站点可打开（首页、写作、概要、圣经、参考库、设置）
- 深色/浅色切换正常（无白块、文字可读）

### 1. 数据安全

- **导出备份 zip** 可下载
- 清空数据后可 **从备份恢复（覆盖）**
- 可 **合并导入备份**（生成新作品 id）
- 章节切换/刷新/关闭页面不丢稿（随机抽样 3 次）

### 2. 写作核心

- 新建作品/卷/章
- 章节排序（拖拽/上下移动）正常
- 章内查找/替换正常
- 全书搜索/范围切换正常
- 导入旧书（.txt/.md/.docx）能自动识别章节（抽样验证标题与正文）

### 3. 圣经 / 概要

- 概要页可编辑并持久化（刷新后仍在）
- 圣经页各 tab 可增删改，顺序/筛选符合预期
- 写作页侧栏：本章约束可编辑；术语命中提示可见

### 4. 参考库

- 批量导入参考文本可跑完（看进度条/无卡死）
- 全文检索可用；点击命中能在阅读器定位
- 摘录：创建/编辑/标签/跳转原文/插入正文
- 必要时可重建索引、清空参考库

### 5. AI（若启用）

- 设置页：AI 提供方与密钥保存正常
- **隐私与上传范围**：未同意时云端 AI 禁止调用
- Ollama 可生成；云端提供方（若开启）可生成
- 生成输出进入草稿区；插入/追加/替换符合预期（替换有二次确认）
- 取消/重试可用

### 6. 站点与合规

- `liubaiai.com` → `www.liubaiai.com` 统一重定向生效（Vercel）
- SEO meta（title/description/canonical/og/twitter）存在
- 应用内可访问：隐私政策 `/privacy`、用户协议 `/terms`
